Basic HTML "Hello World" Example

This is a simple HTML page demonstrating the classic "Hello World" program with clean, modern styling.

Hello, World!

The HTML Code for This Page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello World</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

What is this HTML code doing?

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html lang="en"> - The root element of the page, with language set to English
  • <head> - Contains metadata and links to external resources
  • <meta charset="UTF-8"> - Sets the character encoding to UTF-8
  • <title> - Sets the title that appears in the browser tab
  • <body> - Contains all the visible content of the webpage
  • <h1> - A heading element (largest heading level)
  • <p> - A paragraph element for text content

This page is built with pure HTML and CSS. Save this code in a file with a .html extension and open it in any browser.
